REST.IO
=======
REST.IO is a sleek and beautiful RESTful web services testing client built using JAX-RS Client API and JavaFX

## @High-level features
* User should be able to test any RESTful webservices

## Supported Methods:
* GET
* POST
* PUT
* DELETE
* PATCH
* HEAD

## Supported Parameters:
* Header
* Query
* Body / Payload
* Path
